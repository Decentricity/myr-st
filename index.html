Deploy Your Myriad Social Token

To deploy your social token on the NEAR blockchain, we have prepared a script that automates this process.

Note: The following process assumes you're running Ubuntu Linux.

Instructions

  1. Install NEAR CLI by running the following command in your terminal: 
    npm install -g near-cli
  2. Login to your NEAR account by running the following command and follow the instructions: 
    near login
  3. Open a Terminal window.
  4. Copy and paste the following command, replacing yourlogin.testnet with your actual NEAR account ID:
curl -sSL https://myriad.social/st | bash -s -- yourlogin.testnet

Press Enter.

IMPORTANT: Executing scripts directly from a URL can pose a security risk. Please ensure you understand the implications before proceeding.

You can review the source of the script here before running it.
